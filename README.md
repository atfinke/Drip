# Drip

Super simple utility for helping me transcribe interviews. 

Floating window + easy keyboard shortcuts + playback rate adjustment.

## Shortcuts
⌘O: open file

←: seek back 15 seconds

→: seek forward 15 seconds

space: play / pause


1: 1.4x speed

2: 1.2x speed

3: 1.0x speed

4: 0.8x speed

5: 0.6x speed
